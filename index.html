⚡ GitHub Actions Setup
  • 01 Create a GitHub repo named gaming-laptop-tracker and enable GitHub Pages (branch: main, folder: root)
  • 02 Add this index.html to the root of the repo
  • 03 Add your scraper Python files under scrapers/ folder
  • 04 Add .github/workflows/scrape.yml with hourly cron schedule
  • 05 Add GitHub Secrets: DATABRICKS_HOST, DATABRICKS_TOKEN, TELEGRAM_TOKEN, TELEGRAM_CHAT_ID
  • 06 Trigger Actions manually once to verify scraper runs correctly
  • 07 Dashboard auto-refreshes every hour via Actions — prices appear here automatically
